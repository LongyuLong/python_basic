** 스타일 시트 **
        HTML 문서에 다양한 레이아웃과 꾸미기를 위한 표현적인 언어 CSS 사용
        서식: 선택자(selector){속성: 값; ...}
        선택자 종류는
        1) 요소명: body tag 내의 특정 tag를 직접 지적해서 선택
        2) . : class 선택자 - 클래스 속성값으로 보유하고 있는 요소를 선택
        3) # : id 선택자로 단수 선택 가능

        스타일은 하위 요소에 상속된다.

        CSS 적용 방법: inline, internal, external
*** CSS 적용하기 ***
inline 방식(해당 요소 직접 스타일 지정-span)

(해당 요소 직접 스타일 지정-div)
